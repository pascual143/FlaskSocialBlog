# FlaskSocialBlog

   Application to run a Blog
   
   Features
   
      - Registration Form
      - Log In, Log Out
      - New Posts
      - See the own posts and the posts of the others when log in
      - About page
      - Home page
     
     
# Application Developed with

      - Python
      - Flask
      - SQLAlchemy
      - SQLite
      - Flask_wtf(flaskForm)
      
# This app was made for learning purpose

# To run the app 

        1) Create the environment (env for flask)
        2) Python app.js
